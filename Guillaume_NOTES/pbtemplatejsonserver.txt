# 🧩 **PureBasic TLSv2 Server Template**  
Works with your `tlsv2_server_run()` and callback hooks.

```purebasic
; ============================================================
;  PureBasic TLSv2 Server Template
;  Works with tls_wrapper_v2.c (callbacks + JSON framing)
;  No .l used — only .i, .q, .s
; ============================================================

EnableExplicit

#MAX_JSON = 65536

; ------------------------------------------------------------
; Callback: Client connected
; ------------------------------------------------------------
ProcedureC OnClientConnected(clientID.i)
  Debug ">>> Client connected: " + Str(clientID)
EndProcedure

; ------------------------------------------------------------
; Callback: Client disconnected
; ------------------------------------------------------------
ProcedureC OnClientDisconnected(clientID.i)
  Debug "<<< Client disconnected: " + Str(clientID)
EndProcedure

; ------------------------------------------------------------
; Callback: JSON received
; ------------------------------------------------------------
ProcedureC OnJSONReceived(clientID.i, *json, jsonLen.i)
  Protected msg.s

  msg = PeekS(*json, jsonLen, #PB_UTF8)

  Debug "----------------------------------------"
  Debug "Received JSON from client " + Str(clientID)
  Debug msg

  ; --- Dispatch based on "type" ---
  If FindString(msg, ~"\"type\":\"text\"", 1)
    Debug "-> TEXT message"

    ; Echo back
    tlsv2_send_json(clientID, @"{\"type\":\"text\",\"data\":\"PB server received your text\"}", 63)

  ElseIf FindString(msg, ~"\"type\":\"command\"", 1)
    Debug "-> COMMAND message"

    If FindString(msg, ~"\"cmd\":\"PING\"", 1)
      tlsv2_send_json(clientID, @"{\"type\":\"command\",\"cmd\":\"PONG\"}", 36)
    EndIf

  ElseIf FindString(msg, ~"\"type\":\"table\"", 1)
    Debug "-> TABLE request"

    ; Example: return fake SQLite table
    Protected reply.s = ~"{\"type\":\"table\",\"cmd\":\"SELECT_ALL_RESULT\",\"data\":{\"table\":\"users\",\"columns\":[\"id\",\"name\"],\"rows\":[[1,\"Alice\"],[2,\"Bob\"]]}}"
    tlsv2_send_json(clientID, reply, StringByteLength(reply, #PB_UTF8))

  Else
    Debug "-> UNKNOWN message"
  EndIf

EndProcedure

; ------------------------------------------------------------
; Server configuration structure (mirrors C struct)
; ------------------------------------------------------------
Structure TLSV2_ServerConfig
  port.i
  cert_file.i
  key_file.i
  on_client_connected.i
  on_client_disconnected.i
  on_json_received.i
EndStructure

; ------------------------------------------------------------
; Main
; ------------------------------------------------------------
Define cfg.TLSV2_ServerConfig

cfg\port = 8443
cfg\cert_file = @"server.crt"
cfg\key_file  = @"server.key"

cfg\on_client_connected    = @OnClientConnected()
cfg\on_client_disconnected = @OnClientDisconnected()
cfg\on_json_received       = @OnJSONReceived()

Debug "Starting TLSv2 server on port " + Str(cfg\port)

; Blocking loop inside C
tlsv2_server_run(cfg)

Debug "Server stopped."
```

# 🧠 **Why this PB server template is perfect**

### ✔ Mirrors your C callbacks  
The PB procedures map directly to:

- `tlsv2_on_client_connected`
- `tlsv2_on_client_disconnected`
- `tlsv2_on_json_received`

### ✔ UTF‑8 safe  
Uses `PeekS(*json, len, #PB_UTF8)`.

### ✔ No `.l` anywhere  
Everything uses `.i`, `.s`, `.q`.

### ✔ Cross‑platform  
No Windows API.  
Works on Linux, macOS, BSD, Raspberry Pi.

### ✔ JSON dispatch included  
Simple pattern matching, but you can plug in a real JSON parser later.

### ✔ Ready for your SDK  
Drop it into:

```
/sdk/purebasic/server/tls_server.pb
```
